# Servidor api rest

Este proyecto consiste en un servidor API REST escrito en TypeScript, listo para ser utilizado en cualquier momento. Proporciona una base sólida para desarrollar aplicaciones web y servicios que requieran una interfaz de programación de aplicaciones (API) a través de HTTP. El servidor se encuentra configurado y preparado para el desarrollo y despliegue, con rutas definidas y una estructura modular que permite una fácil extensión y personalización.

# Características Destacadas

- Escrito en TypeScript para una tipificación fuerte y más seguro.
- Configuración sencilla y rápida.
- Estructura modular y organizada para la fácil expansión y mantenimiento.
- Documentación clara de los endpoints disponibles.
- Listo para ser desplegado en un servidor o plataforma de alojamiento web.

¡Comienza a utilizar este servidor API REST y ahorra tiempo en la configuración inicial para tus proyectos!

## Requisitos

Asegúrate de tener instalado lo siguiente:

- Node.js: [Descargar e instalar Node.js](https://nodejs.org/)

## Configuración

1. Clona este repositorio:

  ```bash
   git clone https://github.com/tu-usuario/tu-repo.git
  ```

2. Navega al directorio del proyecto:

  ```bash
   cd SERVER_API_REST
  ```

3. Instala las dependencias:

  ```bash
   npm install
  ```

# Uso
 
- Para ejecutar el servidor en modo de desarrollo:
 
  ```bash
   npm run dev:server
  ```
